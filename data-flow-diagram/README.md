# Data Flow Diagram (DFD)

This folder contains the Data Flow Diagram (DFD) for the Airbnb Clone backend project.

The diagram illustrates how data moves between users, backend processes, and database storage.  
It covers core flows such as:

- User authentication  
- Property management  
- Booking operations  
- Payment processing  
- Review management  

See `data-flow.png` for the full diagram.
